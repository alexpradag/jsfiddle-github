Visualize Mobile Network Antennae by Technology

This example displays a visualization of mobile network antennae on a map.  We introduce the concept of the Data Lens REST API query which is used to fetch and process the data, and then demonstrate how the Data Lens JavaScript libraries are used to produce the visualization.

The dataset used for the visualization contains antenna attributes such as its location, direction ("azimuth") and technology.  (Note that the actual data used here is simulated.)  Each antenna symbol is designed as follows:

  • Antennae are displayed as one or more line segments, each of a solid color.
  • Each segment represents a cell.
  • The direction of the segment is the azimuth of the cell.
  • Different colors represent the cell technologies.
  • The dot in the middle of the symbol represents the antenna's location.
The end-user will be able to control the visibility of cells by toggling the cell technology (2G, 3G, 4G) in the viewport.
