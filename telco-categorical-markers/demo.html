Visualize Mobile Network Antennae by Providers

This example displays a visualization of groups of mobile network antennae on a map.  We introduce the concept of the Data Lens REST API query which is used to fetch and process the data, and then demonstrate how the Data Lens JavaScript libraries are used to produce the visualization.

The dataset used for the visualization contains antenna attributes such as its location, provider and technology.  (Note that the actual data used here is simulated.)  Single antennae are represented by colored circles, while groups of antennae ("clusters") are represented by pie chart symbols ("categorical markers").  These are designed as follows:

  • Each single-antenna marker is displayed as a circle, colored by its provider
  • Each cluster marker is displayed as a pie chart, with a black inner circle and colored ring segments around the perimeter
  • Each cluster marker's inner circle includes a number that indicates the number of antennae in the cluster
  • The length of each ring segment indicates the proportion of antennae in the cluster by each provider to the total number of antennae
The end-user will be able to filter the antennae in the viewport by cell technology (2G, 3G, 4G) using a drop-down list.
